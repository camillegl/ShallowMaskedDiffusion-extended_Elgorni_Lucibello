(242 lines, 16 entries) are **not byte-identical** (different SHA-256, and a
524-line unified diff). Every citation key present in `paper/bibliography.bib`

every real citation key actually used by the retained Typst notes

present in `notes/bibliography.bib`. No merge was needed (paper's key set is
already a subset of notes'); `paper/bibliography.bib` is therefore deleted
as the redundant copy, per the rule "if byte-identical, retain notes copy;
if overlapping-but-not-identical with all keys already present, delete the
paper copy without a merge; only merge if a paper-only key exists."
